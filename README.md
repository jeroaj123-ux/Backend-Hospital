Este proyecto fue desarrollado como parte de una tarea, con el objetivo de implementar un sistema básico de gestión de pacientes de hospital utilizando Java, Spring Boot y PostgreSQL.
El enfoque principal fue crear un método POST completamente funcional que permita registrar nuevos pacientes en la base de datos, además tenemos un método GET creado en clase para consultar los registros existentes.


El sistema simula el módulo de pacientes de un hospital, donde se pueden guardar y listar los datos de cada persona atendida.
La información se almacena directamente en una base de datos PostgreSQL conectada al backend por medio de JPA.

Cada paciente tiene los siguientes datos:

 Nombre del paciente

 Edad

 Historial médico

 Tratamiento actual

 Fecha de registro

 Y su respectiva ID la cual se autoincrementa al agregar uno nuevo
